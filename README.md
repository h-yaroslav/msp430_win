## The same goddamn git

To avoid visiting github by some of your id_rsa(s) (thus receiving `Permission denied (publickey)` please hardcode identity file in the `~/.ssh/config`
```bash
Host github.com
	IdentityFile ~/.ssh/id_rsa.gmail
```
